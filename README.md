#cara menjalankan
1. git clone
2. cd laravel
3. cp .env.example .env
4. composer update
5 php artisan key:generate
6. php artisan migrate
7. npm install 
8. split terminal
9. npm run dev
10. php artisan serve
11. generate key
"Sebelum Menjalankan project di atas pastikan XAMPP susah Berjalan (terkhusus untuk MySQL)"
